Справочник "Инвентарные номера ОС"


Справочник Инвентарные номера предназначен для хранения списка инвентарных номеров ОС, НМА и НПА организации, а также дополнительных реквизитов: Заводской номер, Реестровый номер, Номер паспорта, Иной номер и Дата выпуска.

Примечание. Ввод реквизитов Заводской номер, Номер паспорта и Дата выпуска, производившийся ранее в справочнике Основные средства, НМА, НПА, теперь выполняется в справочнике Инвентарные номера и возможен только для объектов ОС, для которых инвентарные номера созданы.

Инвентарные номера для объектов НФА можно создать как до принятия объектов к учету, в справочнике Инвентарные номера, так и в процессе принятия к учету с помощью документа Принятие к учету ОС, НМА, НПА.

Важно!
Присвоение объекту НФА инвентарного номера, независимо от момента его создания, происходит только при принятии объекта НФА к учету документом Принятие к учету ОС, НМА, НПА.

При создании инвентарного номера контролируется уникальность инвентарного номера в организации.

Работа со списком

Справочник Инвентарные номера можно открыть с помощью команды Инвентарные номера ОС, НМА, НПА меню функций раздела Основные средства, НМА, НПА.

В форме списка предусмотрен отбор по реквизитам Организация и Основное средство. Также реализован быстрый поиск по реквизиту Инвентарный номер.

Создание нового элемента

Для добавления в справочник элемента можно нажать на кнопку Создать командной панели формы Инвентарные номера.

Заполнение формы "Формирование инвентарных номеров"

  • Организация – по умолчанию поле заполнено текущей организацией.
  • Шаблон – по умолчанию поле заполнено значением Шаблон инв. номеров бюджетного учета.
  • Основное средство – основное средство, НМА или НПА у которого отсутствует инвентарный номер. Выбирается из справочника Основные средства, НМА, НПА.
  • Счет учета – счет учета основного средства. Выбирается из фрагмента Плана счетов ЕПСБУ, ограниченного счетами для учета ОС, НМА и НПА.
  • КФО – код вида финансового обеспечения.
  • Порядковый номер – заполняется автоматически.

Для завершения процедуры создания инвентарного номера следует нажать на кнопку Создать или Создать и закрыть.

Двойной щелчок мышью по строке справочника Инвентарные номера вызывает меню с командами:

  • Открыть объект ОС,
  • Открыть инвентарный номер.

Использование команды Открыть объект ОС позволяет открыть карточку объекта ОС в справочнике Основные средства, НМА. НПА.

Команда Открыть инвентарный номер вызывает форму Инвентарный номер, предназначенную для заполнения дополнительных реквизитов.

Заполнение формы "Инвентарный номер"


Реквизиты Организация, Основное средство и Инвентарный номер заполнены по умолчанию.

  • Заводской номер – заводской номер ОС.
  • Реестровый номер – номер в реестре федерального (регионального) имущества.
  • Номер паспорта – номер паспорта ОС.
  • Дата выпуска – дата выпуска ОС.
